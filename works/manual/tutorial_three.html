Tutorial Three - Compartments modification

In this tutorial we will learn how to define the different objects of the ship and identify the possible elements that are applicable, such as bulkheads, decks, equipments and compartments.

On the end of this tutorial, you will understand what type of objects that compose the Vessel.js dataformat and how to create them. After this tutorial, you might be equipped wih the following capabilities:

  • Learn how to set up new compartments and tanks;
  • Learn the different types of objects that can be defined in the library (equipment, compartment and tanks)

Interacting with compartments
